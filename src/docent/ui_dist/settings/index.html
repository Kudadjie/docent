Settings

Manage your Docent preferences and data.

Reading

Controls how Docent syncs your reading queue with your reference manager and local paper database.

Reference manager
Docent reads from your library to build the queue. Your data is never modified.
Watch folder
Mendeley watch folder — PDFs dropped here are auto-imported into your library. Set this to the same folder Mendeley monitors.
not set
Queue collection
Name of the collection to sync from. Must exactly match the collection name in your reference manager.
not set
Research output directory
Folder where Studio research outputs are saved. Defaults to ~/docent/research/ if not set.
not set

Studio

Controls which AI model Feynman uses when running deep research and literature review tasks.

Feynman model
LiteLLM model string passed to Feynman — e.g. groq/llama-3.3-70b-versatile or anthropic/claude-sonnet-4-5. Leave empty to use Feynman's built-in default.
not set

API keys

Keys for research backends and paper search. Stored in ~/.docent/config.toml — never sent anywhere except the respective provider.

Search & discovery
Tavily
Web search for research workflows. Free tier: 1,000 calls/month. Get a key at app.tavily.com — no credit card needed.
not set
alphaXiv
Optional — upgrades Search papers and Get paper with AI overviews, topic tags, and GitHub links. Without it, both actions fall back to the free arXiv API automatically. Free key at alphaxiv.org/settings.
not set
Semantic Scholar
Optional — raises API rate limits for scholarly search. api.semanticscholar.org
not set
